# Cadastral Data Query Application

A web-based mapping application for querying cadastral data, electrical infrastructure, and subdivision analysis in Western Australia.


## Features
- API connectivity with Landgate
- Interactive map with cadastral information
- Automatic property frontage detection
- Polygon subdivision and analysis
- Electrical infrastructure visualisation

## Setup

1. **Clone or download this repository**

2. **This version is intended to be pushed as an Azure Static Webpage, and requires you to login to establish a SLIP token.**
